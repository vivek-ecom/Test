STATIC HTML STARTER STORE

WEBSPHERE COMMERCE VERSION 7.0.0.6

Introduction

WebSphere Commerce provides a set of static, navigable HTML pages for the Aurora starter store. These pages showcase all the store pages and their different supported flows. For example, there are different pages to demonstrate an empty shopping cart, a shopping cart with more than 20 items with pagination invoked, and so on. The asset is viewable by opening the HTML files in a browser without the need of a development environment running on a WebSphere Commerce server.

Note: The materials provided in this package are samples only.
